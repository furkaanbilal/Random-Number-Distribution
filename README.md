# Random Number Distribution

This project is part of my Python learning journey.

## Description

This Python project generates random numbers and shows how they are distributed over multiple runs. It helps demonstrate how randomness works and how values appear with different frequencies.

## Features

- Generates random numbers using Python
- Runs multiple trials
- Counts how often each number appears
- Displays the distribution clearly

## Technologies Used

- Python 3

## How to Run

1. Download or clone this repository
2. Open terminal in the project folder
3. Run:

bash
python main.py


## Purpose

The goal of this project is to practice Python programming and understand random number distribution through code.
